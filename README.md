Steps to Upload Project on GitHub

Create a folder on your computer
Example: Obesity-Prediction-ML

Add your files

obesity_prediction.ipynb

requirements.txt

README.md (use the one I gave you)

Any dataset or model files (optional)

Open the folder in VS Code or Terminal

Run these commands

git init
git add .
git commit -m "Initial commit: Obesity Prediction using Machine Learning"
git branch -M main
git remote add origin https://github.com/yourusername/obesity-prediction-ml.git
git push -u origin main


Go to GitHub → New Repository

Name it obesity-prediction-ml

Don’t add a README (you already have one)

Click Create Repository

Refresh your GitHub repo page
✅ Your project with README will appear there!
